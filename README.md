# 🎵 Moody Player — Mood-Based Music Recommender (Face Expression Detection)

Moody Player is a full-stack web application that detects a user’s facial expression using their device camera and instantly recommends songs based on their mood. The project combines computer vision, real-time webcam access, and a scalable backend API to create a fun and interactive music discovery experience.

# 🚀 Key Features

- 😊 Real-time facial expression detection using face-api.js
- 📷 Works on desktop & mobile cameras (HTTPS enabled)
- 🎧 Mood → Music recommendation system via backend API
- 🌐 Fully deployed system
- 🔒 Secure environment variables (no secrets exposed)
- 📱 Responsive UI for mobile & desktop users
- 🔐 HTTPS enabled locally using mkcert for camera permissions
-  Frontend → Vercel
-  Backend → Render

# 🧠 How It Works

- User opens the website and allows camera access

- Face-API detects the dominant facial expression

- The detected mood is sent to the backend API

- Backend returns curated songs for that mood

- Songs are displayed instantly in the UI 🎶

# 🛠 Tech Stack
- Frontend

- React.js

- Face-API.js

- Axios

- Backend

- Node.js

- Express.js

- REST API

- Environment Variables (API keys)

- HTTPS local development using mkcert
